The eigenspace corresponding to an eigenvalue \(v\) is the set of vectors which have eigenvalue \(v\text{.}\) It is a vector subspace of the vector space on which \(A\) acts. The eigenspace corresponding to the matrix \(A\) would be all vectors that have an eigenvalue for that matrix.
