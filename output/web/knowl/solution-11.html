Solution 1.8.2.1.

\begin{equation*} P(0) = (\braket{0|\psi})^2 = (\bra{0} \cdot \frac{1}{\sqrt{3}} (\ket{0} + \sqrt{2}\ket{1}))^2 = (\frac{1}{\sqrt{3}} \braket{0|0} + \frac{\sqrt{2}}{\sqrt{3}} \braket{0|1})^2 = (\frac{1}{\sqrt{3}} + 0)^2 = \frac{1}{3} \end{equation*}
\begin{equation*} P(1) = (\braket{1|\psi})^2 = (\bra{1} \cdot \frac{1}{\sqrt{3}} (\ket{0} + \sqrt{2}\ket{1}))^2 = (\frac{1}{\sqrt{3}} \braket{1|0} + \frac{\sqrt{2}}{\sqrt{3}} \braket{1|1})^2 = (0 + \frac{\sqrt{2}}{\sqrt{3}})^2 = \frac{2}{3} \end{equation*}
