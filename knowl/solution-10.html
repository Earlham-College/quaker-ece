Solution 1.8.1.1.

\begin{equation*} P(+) = (\braket{+|\psi})^2 = (\frac{1}{\sqrt{2}}(\bra{0} + \bra{1}) \cdot \frac{1}{\sqrt{2}}(\ket{0}-\ket{1}))^2 = (\frac{1}{2}(\braket{0|0} - \braket{0|1} + \braket{1|0} - \braket{1|1}))^2 = (\frac{1}{2}(1 - 0 + 0 - 1))^2 = 0 \end{equation*}
\begin{equation*} P(-) = (\braket{-|\psi})^2 = (\frac{1}{\sqrt{2}}(\bra{0} - \bra{1}) \cdot \frac{1}{\sqrt{2}}(\ket{0}-\ket{1}))^2 = (\frac{1}{2}(\braket{0|0} - \braket{0|1} - \braket{1|0} + \braket{1|1}))^2 = (\frac{1}{2}(1 - 0 - 0 + 1))^2 = 1 \end{equation*}
